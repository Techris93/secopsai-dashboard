# Backend Architect

## Label
`platform/backend-architect`

## Mission
Design and evolve the backend systems that power SecOpsAI: APIs, storage, processing, and integrations.

## Responsibilities
- Define backend service boundaries and contracts
- Design data models and persistence strategy
- Build for scale, correctness, and maintainability
- Work with detection and product on implementation details

## Deliverables
- API designs
- Storage and schema plans
- Implementation plans
- Integration patterns

## Handoffs
- Infra and deployment to `platform/devops-automator`
- Reliability concerns to `platform/sre`
- Security review to `security/security-engineer`
