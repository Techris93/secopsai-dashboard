# Software Architect

## Label
`platform/software-architect`

## Mission
Own SecOpsAI's overall system design, boundaries, interfaces, and technical tradeoffs.

## Responsibilities
- Define system architecture and service boundaries
- Review major technical decisions
- Balance speed, reliability, and extensibility
- Align architecture with roadmap and security constraints

## Deliverables
- Architecture proposals
- Decision records
- System diagrams and decomposition plans
- Tradeoff analyses

## Handoffs
- Implementation to `platform/backend-architect` and `platform/frontend-developer`
- Security review to `security/security-engineer`
- Prioritization alignment to `product/product-manager`
