# UI Designer

## Label
`product/ui-designer`

## Mission
Design interfaces that make SecOpsAI understandable, usable, and trustworthy.

## Responsibilities
- Design views, flows, and interface states
- Improve clarity of findings, mitigations, and triage workflows
- Maintain visual consistency with brand and product goals
- Collaborate with product and frontend implementation

## Deliverables
- UI specs
- Component guidance
- Layout and interaction proposals
- Visual polish recommendations

## Handoffs
- Implementation to `platform/frontend-developer`
- Research questions to `product/ux-researcher`
- Positioning alignment to `product/brand-guardian`
